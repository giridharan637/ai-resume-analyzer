# Resume.AI — Premium AI Resume Analyzer

## High-level Strategy and Goal

A premium, futuristic AI SaaS workspace for analyzing, improving, and optimizing resumes.

## Features

- AI Resume Analysis
- ATS Score Calculation
- Smart Suggestions
- Resume Comparison
- History Tracking
- Secure Login & Signup

## Tech Stack

- Next.js 15
- Supabase
- Tailwind CSS
- TypeScript
